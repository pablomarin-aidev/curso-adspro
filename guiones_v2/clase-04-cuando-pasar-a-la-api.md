# Clase 4 — Cuándo un negocio ya debería pasar a la API

**Módulo 4 · FUNDAMENTOS · ~5 min**
**Formato:** puros bullets cortos, contenido completo de la clase (alternativa a `guiones/`).

- La API de WhatsApp no es "solo para empresas grandes".
- El salto no depende de la facturación ni de los empleados.
- Depende de señales operativas concretas.

## Lo que aprenderás
- Las 6 señales de que ya se necesita la API.
- Por qué el tamaño no es una de ellas.
- Cómo hacer un cálculo simple de ROI.
- Cómo aplicar un autodiagnóstico tipo scorecard.

## Las 6 señales de que ya se necesita la API
- No hay que cumplirlas todas.
- Con 2 o 3 ya es momento de considerar el salto.
- **1. Alto volumen de chats.**
  - Entran más mensajes de los que se alcanzan a responder.
  - El WhatsApp normal no permite automatizar ni repartir la carga.
- **2. Varias asesoras atendiendo.**
  - Dos o más personas peleándose un celular.
  - Un solo número no se abre en varios dispositivos sin bloquearse.
- **3. Necesidad de automatizar.**
  - Copiar y pegar los mismos mensajes decenas de veces.
  - No hay bienvenidas ni respuestas automáticas nativas.
- **4. Seguimiento de leads.**
  - No se sabe a cuántos clientes se dejó sin respuesta.
  - El seguimiento vive en la memoria de la asesora.
- **5. Campañas pagas que generan conversaciones.**
  - Se invierte en pauta pero no se mide qué campaña vendió.
  - No conecta la conversación con la venta.
- **6. Desorden / pérdida de chats.**
  - Chats sin responder, clientes duplicados.
  - No hay bandeja unificada ni historial.

## El cálculo simple de ROI
- Estimar chats por día.
- Contar cuántas personas los atienden.
- Comparar con la capacidad real de atención.
- Cuando el volumen supera la capacidad, se pierden ventas.
- Regla mental: si el costo de las ventas perdidas supera el costo de la herramienta, la API deja de ser gasto.
- Pasa a ser inversión.

## Recurso: Scorecard "¿Mi negocio ya necesita API?"
- Puntuar cada criterio de 0 a 2, total sobre 14:
  - Chats nuevos por día.
  - Personas atendiendo el WhatsApp.
  - ¿Repites mensajes manualmente?
  - ¿Haces seguimiento ordenado?
  - ¿Inviertes en pauta que llega a WhatsApp?
  - ¿Se te pierden chats sin responder?
  - ¿Un cliente ha reclamado que nadie lo atendió?
- Bajo → aún no se necesita la API.
- Medio → empezar a planear el salto.
- Alto → se necesita la API ahora.

## Para recordar
- La API no depende del tamaño del negocio.
- Depende de señales de dolor operativo.
- Con 2 o 3 señales, ya es momento de considerar el salto.
- El scorecard da una respuesta objetiva.
