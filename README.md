# Curso AdsPRO — WhatsApp para ventas, automatización y conversiones

Curso de **13 clases** (concepto + práctica) sobre vender por WhatsApp con la API oficial de Meta: de los fundamentos a CAPI, proveedores, plataformas con IA y compliance. Producido para **AdsPRO** (con Juan Tamayo / Appsen).

## Cómo está organizado

Cada clase existe en **tres formatos**, uno por carpeta:

| Carpeta | Qué contiene | Para qué |
|---|---|---|
| **`guiones/`** | Contenido a grabar: puntos a decir frente a cámara (Hook → Desarrollo → Demo → Cierre). No es palabra por palabra. | Grabar el video |
| **`guiones_v2/`** | Alternativa a `guiones/`: el contenido completo de `contenido/` reescrito en puros bullets (sin prosa ni tablas), más fácil de leer de un vistazo. | Grabar o repasar cuando el formato de `guiones/` cuesta seguir |
| **`contenido/`** | La info completa y desarrollada de cada clase (conceptos, tablas, checklists, recursos). | Material de referencia / apuntes |
| **`paginas/`** | Copy tipo slide para la página de cada clase: número, subtítulo, descripción, "Lo que aprenderás" (3), tags y un bloque destacado. | Armar las páginas/diapositivas |

Los archivos se llaman `clase-01-…` a `clase-13-…` y son consistentes entre las tres carpetas.

`modulo-completo.md` reúne, en un solo archivo, el contenido íntegro (`contenido/`) de las 13 clases en orden — útil para copy+paste o revisión completa. Se genera a partir de los archivos de `contenido/`; si editas una clase, vuelve a generarlo.
