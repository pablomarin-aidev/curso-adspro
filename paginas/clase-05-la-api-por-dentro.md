# 05 · La API por dentro (sin tecnicismos)

> Módulo 5 · LA API POR DENTRO · ~9 min

**Por qué la app de WhatsApp se te queda corta para vender**

La API es la puerta oficial de Meta para que otros programas hablen con WhatsApp por ti. Aquí la entiendes por dentro, sin código: arquitectura, webhooks, la ventana de 24 horas, las plantillas y cuánto cobra Meta.

## Lo que aprenderás
- **01 ·** Qué permite la API que la app normal no puede.
- **02 ·** Cómo viaja un mensaje: número, BSP, CRM y webhooks.
- **03 ·** Ventana de 24h, plantillas (HSM) y el modelo de precios de Meta.

## Tags
`Webhooks` · `Ventana 24h` · `HSM` · `Precios Meta`

## La analogía
La app normal es cocinar tú mismo un plato a la vez. La API es tener meseros que atienden cientos de mesas al tiempo, mientras tú te enfocas en el negocio.
