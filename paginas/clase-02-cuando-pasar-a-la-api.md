# 02 · Cuándo un negocio ya debería pasar a la API

> Módulo 2 · FUNDAMENTOS · ~7 min

**No es cuestión de tamaño, es cuestión de dolor**

Lo que define el salto a la API no es tu facturación ni tus empleados, sino señales operativas concretas. Aprende a reconocerlas y a medir, con tus propios números, si tu negocio ya cruzó la línea.

## Lo que aprenderás
- **01 ·** Las 6 señales de que tu negocio ya necesita la API.
- **02 ·** El cálculo simple de ROI para saber cuándo el salto se paga solo.
- **03 ·** Un scorecard para responder con objetividad: "¿ya la necesito?".

## Tags
`Señales` · `ROI` · `Scorecard` · `Diagnóstico`

## El punto de quiebre
Cuando el costo de las ventas que pierdes por desorden y demora supera el costo de la herramienta, la API deja de ser un gasto y pasa a ser una inversión que se paga sola.
